# LLFree: Lock- and Log-free Allocator

This repo contains the reimplementation of LLFree in C.

**Related Projects**
- LLFree:     https://github.com/luhsra/llfree-rs
- Benchmarks: https://scm.sra.uni-hannover.de/research/llc-bench

## Usage

Release build
```sh
make all
```

Running unit-tests
```sh
make test
```
